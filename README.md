## Get Started

1. Click the **Start Development** launcher.
2. Install the MCP Server to your agent, IDE, or client.
3. Start asking questions!

```json
{
  "mcpServers": {
    "Courier": {
      "url": "http://localhost:3000/mcp",
      "headers": {
        "API_KEY": "XXX"
      }

  }
}
```
